# Week 2 – Spring Core and Maven

## Overview

This repository contains the implementation of Spring Core and Maven exercises completed as part of the Cognizant Digital Nurture 5.0 Java FSE training program.

The exercises demonstrate the fundamentals of Maven project creation, Spring IoC Container, Dependency Injection, XML-based configuration, and Spring Boot.

---

## Technologies Used

- Java 17
- Spring Framework 6
- Maven 3.x
- VS Code
- XML Configuration

---

## Exercises

### Exercise 1: Configuring a Basic Spring Application

**Objective**
- Create a Maven project.
- Configure Spring Core.
- Define Spring beans using XML.
- Load the Spring IoC Container.

**Implemented**
- Maven Project Creation
- Spring Context Dependency
- applicationContext.xml
- BookService
- BookRepository
- Bean Configuration
- Main Class Execution

---

### Exercise 2: Implementing Dependency Injection

**Objective**
- Implement Dependency Injection using Spring.

**Implemented**
- Setter Injection
- BookService
- BookRepository
- XML Bean Wiring
- Bean Injection using `<property>`

---

### Exercise 4: Creating and Configuring a Maven Project

**Objective**
- Create and configure a Maven project.

**Implemented**
- Maven Project Structure
- pom.xml Configuration
- Dependency Management
- Maven Lifecycle Commands

Commands Used

```bash
mvn compile
mvn test
mvn package
mvn clean
```

### Exercise 5: Configuring the Spring IoC Container

**Objective**
- Configure the Spring IoC Container.
- Manage application beans using XML configuration.

**Implemented**
- Bean Configuration
- IoC Container
- ApplicationContext
- Bean Retrieval using getBean()

---

### Exercise 7: Constructor and Setter Injection

**Objective**
- Implement Constructor Injection.
