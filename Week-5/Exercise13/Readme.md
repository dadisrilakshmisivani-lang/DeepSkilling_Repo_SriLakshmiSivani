# Blogger App

## Description

Blogger App is a React application that demonstrates different ways of implementing **Conditional Rendering** in React.

The application contains three components:

1. Book Details
2. Blog Details
3. Course Details

Users can navigate between different sections using buttons, and the respective component is displayed dynamically.

---

## Features

### Book Details Component
- Displays book information.
- Shows availability status using the **Ternary Operator**.

### Blog Details Component
- Displays blog information.
- Uses **If-Else Conditional Rendering** to show different messages based on login status.

### Course Details Component
- Displays course information.
- Uses **If-Else statements** for enrollment status.

### Navigation
- Buttons are provided to switch between:
  - Book Details
  - Blog Details
  - Course Details
