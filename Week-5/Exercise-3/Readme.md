# Exercise 3 – Student Score Calculator

## Objective

Develop a React application that calculates and displays the average score of a student using a functional component.

---

## Components Created

- CalculateScore (Functional Component)

---

## Features

- Accepts student details as props:
  - Name
  - School
  - Total Marks
  - Goal (Number of Subjects)
- Calculates the average score.
- Displays all student details.
- Applies CSS styling using an external stylesheet.

---

## Project Structure

```
ScoreCalculatorApp/
│
├── Components/
│   └── CalculateScore.jsx
│
├── Stylesheets/
│   └── mystyle.css
│
└── App3.jsx
```

---

## Technologies Used

- React
- JavaScript (JSX)
- CSS
- Vite

---

## Steps Performed

1. Created the `ScoreCalculatorApp` folder.
2. Developed the `CalculateScore` functional component.
3. Passed student information using props.
4. Calculated the average score.
5. Applied external CSS styling.
6. Rendered the component through `App3`.
7. Executed the application using `npm run dev`.





---

## Conclusion

This exercise demonstrates how React functional components receive data through props, perform simple calculations, and display formatted output while using external CSS for styling.
