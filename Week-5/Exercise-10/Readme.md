# Office Space Rental App

## Objective

This React application demonstrates the use of JSX, objects, arrays, conditional styling, and rendering data dynamically.

## Features

- Displays a heading using JSX.
- Displays an office image.
- Uses an object to display office details.
- Uses an array of office objects.
- Displays office details using the `map()` method.
- Applies conditional CSS:
  - Rent below ₹60,000 → Red
  - Rent above ₹60,000 → Green

## Technologies Used

- React
- JavaScript (ES6)
- JSX
- CSS
- Vite

## How to Run

```bash
npm install
npm run dev
```

Open:

```
http://localhost:5173
```
