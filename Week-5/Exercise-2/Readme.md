# Exercise 1 – Student Management Portal using React

## Objective

Create a React application named **StudentApp** and implement three reusable components:
- Home
- About
- Contact

Each component displays a welcome message, and all components are rendered from the main application.

---

## Components Created

### Home Component
Displays:

> Welcome to the Home page of Student Management Portal

### About Component
Displays:

> Welcome to the About page of the Student Management Portal

### Contact Component
Displays:

> Welcome to the Contact page of the Student Management Portal

---

## Project Structure

```
StudentApp/
│
├── src/
│   ├── StudentApp/
│   │   ├── Home.jsx
│   │   ├── About.jsx
│   │   ├── Contact.jsx
│   │   └── App1.jsx
│   ├── App.jsx
│   ├── main.jsx
│   ├── App.css
│   └── index.css
│
├── public/
├── package.json
└── vite.config.js
```

---

## Technologies Used

- React
- Vite
- JavaScript (JSX)
- HTML
- CSS

---

## Steps Performed

1. Created a React project using Vite.
2. Created a `StudentApp` folder inside the `src` directory.
3. Developed three reusable components:
   - Home
   - About
   - Contact
4. Created `App1.jsx` to render all three components.
5. Updated `App.jsx` to render `App1`.
6. Executed the application using:

```bash
npm run dev
```

---

## Output

The application displays the following messages:

- Welcome to the Home page of Student Management Portal
- Welcome to the About page of the Student Management Portal
- Welcome to the Contact page of the Student Management Portal
