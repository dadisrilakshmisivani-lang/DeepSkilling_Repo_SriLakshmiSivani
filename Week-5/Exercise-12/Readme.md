# Ticket Booking App

## Description

Ticket Booking App is a React application where users can view flight details and book tickets.

The application provides two types of access:

- **Guest User**: Can browse available flight details.
- **Logged-in User**: Can book flight tickets after login.

The application uses React state management to handle login and logout functionality.

---

## Features

### Guest Page
- Displays available flight details.
- Guest users can view:
  - Airline name
  - Source location
  - Destination location
  - Ticket price
- Guest users cannot book tickets.

### Login Page
- Allows users to login.
- After successful login, the user is redirected to the User Page.

### User Page
- Displays user-specific content.
- Allows logged-in users to book tickets.
- Provides logout option.

### Logout
- Logs out the user.
- Redirects back to the Guest Page.
