# Week 1 - Design Patterns and Principles,Data Structures and Algorithms

This repository contains the exercises completed during Week 1 of the Digital Nurture Program.

## Exercises Completed

### Exercise 1: Implementing the Singleton Pattern

Implemented the Singleton Design Pattern to ensure that only one instance of a class is created and accessed throughout the application.

### Exercise 2: Implementing the Factory Method Pattern

Implemented the Factory Method Design Pattern to create objects through a factory interface without directly instantiating concrete classes.
 

### Exercise 2(DSA): E-commerce Platform Search Function

Developed a search functionality for an e-commerce platform using searching algorithms to efficiently locate products.

### Exercise 7(DSA): Financial Forecasting

Implemented a recursive solution to forecast future financial values based on historical growth rates.

### Exercise 1: Control Structures
